# Tabungan Ceria

Sistem Tabungan Sekolah Berbasis Online - Aplikasi web untuk mengelola tabungan siswa secara kolektif dengan integrasi Google Sheets sebagai database cloud.

## Fitur

- Portal Kepala Sekolah: Dashboard pengawasan dan audit seluruh kelas
- Portal Wali Kelas: Manajemen siswa dan transaksi per kelas
- Portal Orang Tua/Wali: Pantau saldo dan riwayat transaksi anak
- Integrasi Google Sheets: Sinkronisasi data real-time ke cloud
- Ekspor Laporan: CSV dan PDF

## Menjalankan Aplikasi

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Jalankan aplikasi:
   `npm run dev`

Aplikasi akan berjalan di http://localhost:3000
